## 最新版本摘要（v0.1.4 / 2025-10）

### 本次重點
- 建置穩定：改用 webpack、移除 Google Fonts 依賴、加入 `force-dynamic` 避免 build 期外網需求。
- Next.js 15 相容性：`params`/`searchParams` 改為 Promise 介面並調整使用方式。
- 型別與程式穩定：統一 filter 與斷言寫法、修正章節表單/資料清洗型別。
