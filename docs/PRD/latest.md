## 最新版本摘要（v0.2 / 2025-10）

### 本次重點
- **章節模組化**：課程資料全面使用 `modules` 結構儲存章節與課程清單，後台表單可管理排序、影片網址、免費試看與影片時長；API 會自動同步扁平 `syllabus` 以及更新課程總時數 / 堂數。
- **學習體驗升級**：  
  - `/courses/[id]` 依學員是否購買顯示不同 CTA，已購買者可直接播放所有章節影片並顯示「已完成 / 下一堂課」狀態。  
  - `/learning` 提供章節列表、完成統計與前往課程章節的快速連結。
- **Demo 警示**：首頁 Hero 區塊新增紅色警示 Banner，明確標示本站為練習作品。
- **文件同步**：README 及 PRD 更新章節資料結構、手動測試建議與產品概況。

### 重要頁面狀態
| 頁面 | 最新狀態 |
| --- | --- |
| `/courses/[id]` | 章節展開時顯示章節徽章、影片試看與正式課程提示；已購會員 CTA 變為「開始學習」。 |
| `/learning` | 顯示章節完成度、下一堂課、前往章節連結，並更新最後學習時間。 |
| `/admin/courses`、`/instructor/courses` | 使用新版 `CourseForm` 編輯章節，送出時同步 modules / syllabus。 |
| `/` | Hero 區塊新增 Demo 警示徽章。 |

> 若需查看舊版需求，可參考 [`v0.1.md`](./v0.1.md)。
