## vibe coding 線上課程平台 PRD

### 平台概述
- 產品性質：以 Next.js 建構的線上課程示範平台，支援課程瀏覽、購買、後台管理與學習進度追蹤。
- 核心角色：
  - 學員：瀏覽課程、購買並在「我的學習」中追蹤章節進度。
  - 講師／管理員：於後台維護課程、講師資訊與訂單資料。
- 最新狀態（v0.1.4）：導入 `modules` 章節結構與學習整合，並完成建置穩定性調整（webpack build、系統字體、動態頁面避免外網依賴）。

### 主要使用流程
1. **瀏覽與引導**  
   - 首頁提供熱門課程卡片，並醒目標示 Demo 站點提示。  
   - `/courses` 透過搜尋與篩選協助用戶快速找課。
2. **課程詳情與購買**  
   - `/courses/[id]` 呈現章節目錄、免費試看影片、講師資訊與購買方案。  
   - 已購學員會看到「開始學習」按鈕並可直接播放所有章節。
3. **購買流程**  
   - `/purchase/[courseId]` 支援折扣碼試算後導向 ATM 匯款流程。  
   - `/checkout/[courseId]/atm` 填寫匯款完成後即寫入 `enrollments`，並回到 `/learning`。
4. **學習體驗**  
   - `/learning` 顯示章節列表、下一堂課提示與完成標記，連回課程詳情對應章節。  
   - 課程詳情頁會同步更新最後觀看時間與完成進度。
5. **課程維運**  
   - `/admin/courses` 與 `/instructor/courses` 以新版 `CourseForm` 維護章節、影片與試看設定。  
   - `/api/admin/courses` 建立／更新課程時同步寫入 `modules` 與扁平 `syllabus`，並自動換算時數與堂數。
